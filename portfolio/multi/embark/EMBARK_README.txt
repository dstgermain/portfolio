Embark READ ME

Embark is a game set in the future, where the featured ship is called "Embark". It is headed to a distant planet when it come into an asteroid filed. 

Avoid asteroids or shoot them down, the object is to not get hit while shooting as many asteroids as possible. Be sure to be as accurate as possible, accuracy is calculated into the final score.

Movement: use the arrow keys on your keyboard to move the ship around the screen and the spacebar to fire the ships blasters.

Sound: Turn sound off on the options screen. The game has sound effects as well as music.

Difficulty: Change the difficulty settings in the options menu
